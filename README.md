# 🎮 Taş Kağıt Makas Oyunu (Python)

Bu proje, Python öğrenme sürecimde geliştirdiğim basit ama işlevsel bir taş-kağıt-makas oyunudur.
Kullanıcı ile bilgisayar arasında oynanır ve 5 tur üzerinden skor tutulur.

---

## 🎯 Proje Amacı

* Python’da döngü, koşul ve kullanıcı etkileşimini öğrenmek
* Basit bir oyun mantığı kurabilmek
* Algoritmik düşünme becerisini geliştirmek

---

## 🚀 Özellikler

* Kullanıcı vs bilgisayar oyunu
* Rastgele bilgisayar seçimi
* 5 tur üzerinden oyun sistemi
* Beraberlik durumunda tur sayılmaz
* Skor takibi (kullanıcı & bilgisayar)
* Hatalı giriş kontrolü
* Oyun sonunda kazanan belirleme
* Tekrar oynama seçeneği

---

## 🧠 Nasıl Çalışır?

1. Kullanıcı taş, kağıt veya makas seçer
2. Bilgisayar rastgele seçim yapar
3. Seçimler karşılaştırılır
4. Kazanan belirlenir ve skor güncellenir
5. 5 geçerli tur sonunda oyun biter
6. Kullanıcı isterse oyun yeniden başlatılır

---

## ▶️ Nasıl Çalıştırılır?

1. Python yüklü olmalıdır
2. Terminal veya komut satırında:

```bash id="f98s1k"
python tas_kagit_makas.py
```
